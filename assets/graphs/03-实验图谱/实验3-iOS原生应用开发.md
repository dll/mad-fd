# 实验3：iOS原生应用开发
*基于刘东良教师教学大纲制定*

## 实验基本信息
- **实验性质**：设计性实验
- **实验学时**：4学时
- **适用专业**：计算机科学与技术
- **先修知识**：Swift语言基础、面向对象编程、实验1环境搭建
- **实验平台**：macOS + Xcode

## 实验目标
- 掌握iOS应用开发基本流程和架构
- 熟练使用Xcode开发工具和Interface Builder
- 理解iOS应用生命周期和设计模式
- 完成一个完整的iOS原生应用
- **重点**：在iOS开发中深度应用AI编程工具

## 实验项目：任务管理iOS应用

### 项目需求分析
开发一个"任务管理"iOS应用，具备以下功能：

#### 核心功能模块
1. **任务列表功能**
   - 任务列表显示
   - 分类和优先级标识
   - 完成状态切换
   - 搜索和筛选

2. **任务创建和编辑**
   - 任务标题和描述
   - 截止日期设置
   - 优先级选择
   - 分类标签管理

3. **任务详情查看**
   - 详细信息展示
   - 子任务支持
   - 进度跟踪
   - 备注和附件

4. **数据同步和备份**
   - 本地数据持久化
   - iCloud同步支持
   - 数据导入导出
   - 删除恢复功能

### 技术实现要点

#### UI设计要求
- **主界面设计**
  - UITableView任务列表

- **编辑界面设计**
  - 表单式编辑界面
  - 日期选择器集成

#### 数据管理方案

- **iCloud集成**
  - CloudKit框架使用
