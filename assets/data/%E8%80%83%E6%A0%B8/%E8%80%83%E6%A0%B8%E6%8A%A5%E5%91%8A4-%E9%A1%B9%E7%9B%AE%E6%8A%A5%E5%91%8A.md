### 课程考核成绩汇总

**🔴 重要警告：四份报告缺一不可**

> **⚠️ 必须四个考核报告均提交，考核成绩才有效。缺少任何一份报告，大作业成绩为0分！**

**必须提交的四份报告**：
1. ✅ 考核报告1：答辩报告（演示答辩）
2. ✅ 考核报告2：个人报告
3. ✅ 考核报告3：小组报告
4. ✅ 考核报告4：项目报告

**后果说明**：
- ❌ 缺少1份报告：大作业成绩0分（占课程总成绩50%）
- ❌ 缺少2份或以上：大作业成绩0分
- ❌ 迟交任何一份报告：按缺交处理，成绩0分
- ⚠️ 四份报告必须在规定时间内全部提交

---

**📌 课程总成绩构成**：

**大作业考核占课程总成绩的50%**，四个考核报告各占1/4，具体如下：

| 考核项目 | 占课程总成绩比例 | 占大作业成绩比例 | 满分 | 得分 | 备注 |
|---------|----------------|----------------|------|------|------|
| **考核报告1：演示答辩** | 12.5% | 25% | 100分 |      | 三个必答题+随机题，视频演示，源码+部署 |
| **考核报告2：个人报告** | 12.5% | 25% | 100分 |      | 四阶段工作+三类图表（类图、顺序图、架构图） |
| **考核报告3：小组报告** | 12.5% | 25% | 100分 |      | 四阶段综合+个人贡献，每人独立整合 |
| **考核报告4：项目报告** | 12.5% | 25% | 100分 |      | 技术文档+部署+创新，每人独立整合 |
| **大作业总分** | **50%** | **100%** | **400分** | **___** | 四项总分 |

**成绩计算公式**：
```
大作业总分 = (演示答辩得分 + 个人报告得分 + 小组报告得分 + 项目报告得分) / 4

课程总成绩中的大作业部分 = 大作业总分 × 50%
```

**示例**：
- 如果演示答辩85分、个人报告90分、小组报告88分、项目报告92分
- 大作业总分 = (85 + 90 + 88 + 92) / 4 = 88.75分
- 占课程总成绩 = 88.75 × 50% = 44.375分（满分50分）
