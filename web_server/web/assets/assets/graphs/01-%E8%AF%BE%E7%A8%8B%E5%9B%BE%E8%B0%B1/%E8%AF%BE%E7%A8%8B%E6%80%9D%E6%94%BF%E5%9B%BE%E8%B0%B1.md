# 0102 移动应用开发课程思政图谱
*基于刘东良教师教学大纲制定*

## 课程思政总体目标
将立德树人根本任务融入移动应用开发课程教学全过程，在传授技术知识的同时，培养学生的科技报国情怀、工匠精神、创新意识和社会责任感，为培养德才兼备的高素质计算机专业人才服务。

## 1. 价值引领与思政元素

### 010201 科技自立自强精神
- **民族科技自信培养**
  - 通过华为鸿蒙生态发展案例，培养民族品牌自信与技术创新精神
  - 分析华为在操作系统领域的突破，增强学生对自主创新的认识和支持
  - 微信小程序、支付宝小程序等自主技术平台的成功案例分析
  - 国产移动应用在国际市场的竞争优势和发展前景

- **科技报国情怀**
  - 移动互联网对国家数字化转型的战略意义
  - 5G、物联网等前沿技术在国家建设中的重要作用
  - 移动应用开发人才在数字经济发展中的责任与使命
  - 通过技术服务国家重大战略的职业理想教育

### 010202 工匠精神培养
- **精益求精的技术追求**
  - 从系统底层逻辑培养严谨的工程思维与问题溯源能力
  - 通过原生开发的系统性学习，培养学生对技术细节的追求和对代码质量的坚持
  - 代码规范、注释完整、测试充分的职业习惯养成
  - 性能优化、用户体验提升的持续改进意识

- **技术创新精神**
  - 通过项目迭代过程，培养精益求精的工匠精神与团队协作意识
  - 强调代码质量的重要性，培养学生对技术的敬畏之心和持续改进的态度
  - AI编程工具的合理使用，平衡效率与质量的关系
  - 技术选型决策中的全局思维和长远考虑

### 010203 创新意识与创业精神
- **技术创新思维**
  - 通过"一次开发多端运行"案例，培养效率意识与系统思维
  - 引导学生思考技术选型对项目成功的重要性，培养全局观念和优化思维
  - 跨平台技术发展趋势的前瞻性思考
  - 新技术在解决实际问题中的创新应用

- **创业实践能力**
  - 移动应用创业案例分析，激发学生创新创业热情
  - 技术产品化、商业化的思维培养
  - 用户需求分析与市场洞察能力
  - 团队组建与项目管理经验积累

## 2. 社会责任与职业素养

### 010204 技术伦理与社会责任
- **数据安全与隐私保护**
  - 用户隐私保护的法律法规和技术实现
  - 移动应用中数据收集、存储、使用的合规性
  - GDPR、网络安全法等法规在开发中的体现
  - 安全漏洞的社会危害和开发者责任

- **技术普惠与社会服务**
  - 结合小程序在政务、公益领域的应用案例，培养技术服务社会的价值观
  - 分析小程序如何让数字服务更加便民惠民，增强学生的社会责任感
  - 无障碍应用设计，服务残障人士等特殊群体
  - 智慧医疗、在线教育等民生应用的社会价值

### 010205 职业道德与诚信品质
- **学术诚信与知识产权**
  - 代码原创性与开源协议的正确理解
  - 抄袭、盗版等行为的法律后果和道德风险
  - 尊重他人智力成果，合理使用开源代码
  - 技术文档、学术论文的诚信撰写

- **职业责任与质量意识**
  - 软件质量对用户安全和体验的重要影响
  - 代码审查、测试验证的职业习惯
  - 版本管理、文档编写的规范要求
  - 持续学习、技能更新的职业发展需求

### 010206 团队协作与沟通能力
- **团队精神与协作意识**
  - 移动应用开发中的团队分工与协作
  - 代码协作、版本控制中的团队规范
  - 跨部门协作中的沟通技巧和换位思考
  - 冲突处理与共识达成的能力培养

- **国际视野与文化自信**
  - 国外优秀移动应用的技术学习与文化思考
  - 中国移动互联网发展模式的独特优势
  - 在技术交流中传播中华优秀文化
  - 构建人类命运共同体理念在技术合作中的体现

## 3. 课程思政实施策略

### 3.1 知识传授中的价值塑造
**第一章：移动应用开发技术体系（4学时）**
- 思政融入点：通过国内外移动生态竞争案例，培养科技自立自强意识与技术判断力品格
- 实施方法：
  - 案例分析：华为鸿蒙vs安卓/iOS的技术路线对比
  - 专题讨论：中国移动互联网发展的"弯道超车"现象
  - 价值引导：技术选型背后的国家战略和文化价值

**第二章：原生开发基础（3学时）**
- 思政融入点：从系统底层逻辑培养严谨的工程思维与问题溯源能力
- 实施方法：
  - 代码规范教育：强调工程师的职业操守
  - 调试技能训练：培养科学严谨的分析方法
  - 文档撰写要求：养成负责任的工作态度

**第三章：混合开发技术（5学时）**
- 思政融入点：通过"一次开发多端运行"案例，培养效率意识与系统思维
- 实施方法：
  - 技术演进史：从重复开发到跨平台的效率提升
  - 权衡决策：性能与效率的平衡艺术
  - 创新案例：国产跨平台框架的技术贡献

**第四章：小程序开发（3学时）**
- 思政融入点：结合小程序在政务、公益领域的应用案例，培养技术服务社会的价值观
- 实施方法：
  - 应用案例：健康码、政务服务等疫情防控和民生服务应用
  - 社会价值：技术如何让生活更美好
  - 责任意识：开发者在数字化社会建设中的使命

**第五章：华为多端应用开发（4学时）**
- 思政融入点：通过鸿蒙生态发展案例，培养民族品牌自信与技术创新精神
- 实施方法：
  - 技术突破：分布式软总线等自主创新技术
  - 生态建设：开发者社区与产业合作
  - 文化自信：中国方案对全球技术发展的贡献

**第六章：综合开发实践（5学时）**
- 思政融入点：通过项目迭代过程，培养精益求精的工匠精神与团队协作意识
- 实施方法：
  - 质量文化：零缺陷、持续改进的理念
  - 团队协作：代码审查、知识分享的文化
  - 社会责任：项目对用户、社会的积极影响

### 3.2 能力培养中的素养提升
**实验教学思政融入**
- 实验1-2（环境搭建、原生开发）：培养严谨的学习态度和解决问题的坚持精神
- 实验3-5（跨平台开发、小程序）：强化创新思维和用户服务意识
- 实验6（华为多端开发）：增强民族科技自信和自主创新精神
- 实验7（综合项目）：全面培养团队协作、质量意识和社会责任感

**AI工具使用的伦理教育**
- 合理使用AI编程工具，避免过度依赖
- 理解AI生成代码的局限性和风险
- 保护知识产权，尊重原创作品
- 培养批判性思维，不盲目信任AI建议

### 3.3 实践活动中的品德养成
**校园服务应用开发项目**
- 项目选题：面向校园师生实际需求，体现服务意识
- 开发过程：注重用户体验，培养人文关怀
- 团队协作：建立有效沟通，培养合作精神
- 成果分享：开源代码贡献，体现奉献精神

**技术社区参与**
- 鼓励学生参与开源项目，培养分享精神
- 撰写技术博客，传播正能量的技术文化
- 参加技术竞赛，在竞争中提升综合素质
- 技术志愿服务，用专业技能服务社会

## 4. 思政考核评价体系

### 4.1 考核内容与方式
**平时成绩（20%）中的思政评价**
- 课堂讨论参与度：对思政案例的思考深度和表达能力
- 作业完成质量：代码规范性、文档完整性体现的职业素养
- 学习态度表现：主动学习、互帮互助的品德表现

**实验成绩（30%）中的思政评价**
- 团队协作表现：在团队项目中的责任担当和协作精神
- 创新思维体现：在实验中的创新尝试和问题解决思路
- 社会价值导向：实验项目对社会、用户的积极意义

**期末考查（50%）中的思政评价**
- 项目社会价值：所开发应用的社会意义和用户价值
- 技术伦理体现：在项目中对数据安全、隐私保护的考虑
- 文档质量水平：技术文档的规范性和责任心体现

### 4.2 思政教育效果评估
**知识内化程度**
- 对科技创新、民族自信等理念的理解深度
- 对职业道德、社会责任的认知水平
- 对团队协作、工匠精神的践行能力

**行为外化表现**
- 学习过程中的自主性和责任感
- 团队协作中的奉献精神和领导力
- 技术应用中的伦理意识和社会关怀

**价值观塑造效果**
- 科技报国的理想信念
- 精益求精的职业品格
- 服务社会的责任担当
