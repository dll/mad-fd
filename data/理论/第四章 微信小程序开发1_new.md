## 学前测验：微信小程序基础概念检测

### 测验说明
本测验旨在检测学习者对微信小程序基础概念的了解程度，帮助确定学习起点。

### 第1题
**题目**：微信小程序的技术框架是？

A. React Native
B. Flutter
C. MINA
D. Ionic

**正确答案**：C

**解释**：MINA是微信小程序的技术框架，提供了小程序的运行环境和API，包含了视图层和逻辑层的分离架构。

### 第2题
**题目**：小程序的WXML相当于网页开发中的什么语言？

A. HTML
B. CSS
C. JavaScript
D. Python

**正确答案**：A

**解释**：WXML是微信小程序标记语言，类似于HTML，用于描述小程序页面的结构。

### 第3题
**题目**：小程序页面的生命周期函数中，哪个是页面加载时调用？

A. onShow
B. onLoad
C. onReady
D. onHide

**正确答案**：B

**解释**：onLoad是页面加载时调用的生命周期函数，适合进行页面初始化和数据请求。

### 第4题
**题目**：小程序中用于样式定义的文件格式是？

A. WXML
B. WXSS
C. JS
D. JSON

**正确答案**：B

**解释**：WXSS是微信小程序样式表，类似于CSS，用于定义页面元素的样式。

### 第5题
**题目**：小程序云开发不包括以下哪个能力？

A. 云函数
B. 云数据库
C. 云存储
D. 云推送

**正确答案**：D

**解释**：小程序云开发包括云函数、云数据库和云存储三大能力，不包括云推送。

### 测验结果评估
- **5题全对**：优秀！您对小程序开发有很好的基础
- **3-4题正确**：良好！您具备一定基础
- **1-2题正确**：需要加强！建议先补充小程序基础概念
- **0题正确**：建议先学习小程序入门知识

### 幻灯片3：MINA框架结构
- **标题**：小程序运行架构
- **内容**：
  - **双线程模型**：
    - 逻辑层：AppService运行JS
    - 渲染层：WebView渲染WXML
    - 通信：Native桥接层
  - **文件结构**：
    ```
    pages/
      index/
        index.js      # 逻辑
        index.json    # 配置
        index.wxml   # 结构
        index.wxss   # 样式
    app.js              # 全局逻辑
    app.json            # 全局配置
    app.wxss            # 全局样式
    ```
  - **工作原理**：
    - 开发者编写代码
    - 微信客户端渲染
    - Native处理能力调用

### 幻灯片4：WXML语法基础
- **标题**：小程序标记语言
- **内容**：
  - **数据绑定**：
    ```xml
    <view>Hello {{name}}</view>
    <view>{{flag ? '是' : '否'}}</view>
    ```
  - **列表渲染**：
    ```xml
    <view wx:for="{{items}}" wx:key="id">
      {{index}}: {{item.name}}
    </view>
    ```
  - **条件渲染**：
    ```xml
    <view wx:if="{{show}}">显示内容</view>
    <view wx:else>隐藏内容</view>
    ```
  - **模板引用**：
    ```xml
    <import src="item.wxml"/>
    <include src="header.wxml"/>
    ```

### 幻灯片5：WXSS样式基础
- **标题**：小程序样式语言
- **内容**：
  - **选择器**：
    ```css
    .container { }      /* 类选择器 */
    #title { }          /* ID选择器 */
    view { }            /* 元素选择器 */
    view.active { }     /* 组合选择器 */
    ```
  - **常用属性**：
    - flex布局
    - 尺寸单位：rpx
    - 样式导入
  - **响应式适配**：
    - rpx自动转换
    - 750rpx = 屏幕宽度

### 幻灯片7：页面生命周期
- **标题**：页面生命周期管理
- **内容**：
  - **生命周期函数**：
    - onLoad：页面加载

    - onReady：页面初次渲染完成
    - onHide：页面隐藏
    - onUnload：页面卸载
  - **执行顺序**：
    onLoad → onShow → onReady → onHide/onUnload
