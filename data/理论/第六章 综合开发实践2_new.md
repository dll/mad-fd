学前测验：Git协作与AI应用基础认知

### 测验说明
本测验检测学习者对Git版本控制、团队协作、AI辅助开发等概念的认知，帮助确定本章进阶学习起点。

### 第1题
**题目**：Git中用于创建新分支的命令是？

A. git branch
B. git checkout
C. git commit
D. git merge

**正确答案**：A

**解释**：git branch命令用于创建新分支，git checkout用于切换分支，git commit用于提交，git merge用于合并分支。

### 第2题
**题目**：代码审查的主要目的是？

A. 增加代码量
B. 提高代码质量和发现错误
C. 减少开发时间
D. 加快发布速度

**正确答案**：B

**解释**：代码审查的主要目的是通过团队成员互相检查代码，提高代码质量，发现潜在错误和缺陷，促进知识共享。

### 第3题
**题目**：AI编程工具的主要作用不包括？

A. 代码生成
B. 性能测试
C. 代码优化建议
D. 错误诊断

**正确答案**：B

**解释**：AI编程工具主要用于代码生成、优化建议和错误诊断，性能测试不是其主要作用，需要使用专门的性能测试工具。

### 第4题
**题目**：Git中用于将代码推送到远程仓库的命令是？

A. git pull
B. git push
C. git fetch
D. git clone

**正确答案**：B

**解释**：git push用于将本地代码推送到远程仓库，git pull用于拉取并合并，git fetch用于拉取但不合并，git clone用于克隆仓库。

### 第5题
**题目**：关于AI辅助开发，以下说法正确的是？

A. AI可以完全替代程序员
B. AI生成的代码不需要人工审查
C. AI是辅助工具，需要人工监督
D. AI不适合移动开发

**正确答案**：C

**解释**：AI是辅助工具，可以帮助提高开发效率，但生成的代码需要人工审查和验证，不能完全替代程序员的工作。

    ```bash
    git clone <url>         # 克隆仓库

    git add .               # 暂存文件
    git commit -m "消息"    # 提交
    git push                # 推送到远程
    git pull                # 拉取合并
    ```
  - **分支操作**：
    ```bash
    git branch              # 查看分支
    git branch <name>       # 创建分支

    git merge <branch>     # 合并分支
    ```
