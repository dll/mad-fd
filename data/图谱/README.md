# 移动应用开发知识图谱

## 📋 图谱概述

这是一个完整的移动应用开发知识图谱体系，涵盖了从基础理论到实践项目的全方位内容，适用于高等院校移动应用开发课程教学和企业技术培训。

## 🚀 快速开始

### 1. 🔍 浏览图谱
推荐使用图谱可视化工具查看知识结构：
```bash
cd ../图谱工具项目
python main.py
```

### 2. 📖 选择路径
根据你的背景选择学习路径：
- **零基础**：从`01-课程图谱/知识体系图谱.md`开始系统学习
- **教师**：参考`05-教学图谱`进行课程设计和教学
- **学生**：使用`06-学习图谱`制定个人学习计划
- **有编程基础**：直接进入`02-技术栈图谱`选择感兴趣的技术
- **实践为主**：从`03-实验图谱`开始动手实践

### 3. 🎯 设定目标
明确学习目标：
- **学生**：按照学习图谱系统掌握知识和技能
- **教师**：利用教学图谱提升教学质量和效果
- **求职者**：重点掌握主流技术栈和项目经验
- **在职提升**：针对性学习新技术和提升能力

### 4. 📝 实践项目
通过项目巩固知识：
- **基础项目**：智慧校园生活服务平台（6技术栈集成）
- **进阶项目**：在线学习辅助平台（AI集成教育平台）
- **高级项目**：智能健康运动记录平台（IoT健康平台）
- **创新项目**：二手物品交易平台（社交电商平台）
