# 课程达成度计算系统 - 使用说明

## 系统简介

这是一个基于Python Tkinter开发的图形界面课程达成度计算系统，支持导入Excel/Word文件、可视化展示、生成报告、打印功能，并可配置不同课程的权重，实现通用化设计。

## 系统特点

1. **图形界面操作**：直观的图形界面，无需编程知识
2. **通用化设计**：可配置不同课程的权重，适用于各种课程
3. **完整功能**：导入、计算、可视化、生成、打印一体化
4. **灵活配置**：支持自定义课程名称、班级名称和权重配置

## 功能模块

### 1. 文件导入

#### 导入Excel成绩文件
- 支持格式：.xlsx, .xls
- 自动解析学生数据
- 提取各课程目标达成度

#### 导入Word大纲文件
- 支持格式：.docx, .doc
- 自动解析课程大纲
- 提取课程信息

### 2. 课程配置

#### 课程信息配置
- **课程名称**：输入课程名称（默认：移动应用开发）
- **班级名称**：输入班级名称（默认：软件23+6）

#### 权重配置
- **课程目标1权重**：默认0.15
- **课程目标2权重**：默认0.25
- **课程目标3权重**：默认0.30
- **课程目标4权重**：默认0.30

**注意**：权重总和建议为1.0

### 3. 达成度计算

点击"计算达成度"按钮后，系统将：
1. 计算班级平均达成度
2. 计算加权总达成度
3. 显示计算结果
4. 判定达成度等级

**达成度等级标准**：
- 优秀：≥0.85
- 良好：0.70-0.84
- 中等：0.60-0.69
- 未达成：<0.60

### 4. 可视化功能

#### 生成柱状图
- 展示各课程目标的达成度
- 显示具体数值
- 保存为PNG格式

#### 生成雷达图
- 多维度展示达成度
- 直观对比各目标
- 保存为PNG格式

#### 生成分布图
- 箱线图展示学生分布
- 显示统计信息
- 保存为PNG格式

### 5. 报告生成

#### 生成Markdown报告
- 生成.md格式报告
- 包含完整的分析内容
- 便于编辑和查看

#### 生成Word报告
- 生成.docx格式报告
- 专业排版格式
- 便于打印和分发

### 6. 打印功能

点击"打印报告"按钮后：
1. 自动生成Markdown报告
2. 使用默认浏览器打开报告
3. 使用浏览器的打印功能进行打印

## 使用步骤

### 步骤1：启动系统
```bash
python course_achievement_gui.py
```

### 步骤2：导入数据
1. 点击"导入Excel成绩文件"按钮
2. 选择包含学生成绩的Excel文件
3. 等待数据解析完成

### 步骤3：配置课程（可选）
1. 修改课程名称和班级名称
2. 根据需要调整各课程目标权重
3. 点击"保存配置"按钮

### 步骤4：计算达成度
1. 点击"计算达成度"按钮
2. 查看计算结果
3. 确认达成度等级

### 步骤5：生成可视化
1. 点击"生成柱状图"查看达成度对比
2. 点击"生成雷达图"查看多维度分析
3. 点击"生成分布图"查看学生分布

### 步骤6：生成报告
1. 点击"生成Markdown报告"生成文本报告
2. 点击"生成Word报告"生成专业报告
3. 点击"打印报告"进行打印

## 输出文件

所有生成的文件保存在`./output`目录下：

### 可视化图表
- 课程目标达成度柱状图.png
- 课程目标达成度雷达图.png
- 学生个体达成度分布图.png

### 报告文件
- {班级名称}《{课程名称}》课程达成度报告.md
- {班级名称}《{课程名称}》课程达成度报告.docx

## Excel文件格式要求

Excel文件需要包含以下结构：

1. **表头行**：包含"课程目标"字样的行
2. **数据列**：
   - 学号
   - 姓名
   - 目标1得分
   - 目标1达成度
   - 目标2得分
   - 目标2达成度
   - 目标3得分
   - 目标3达成度
   - 目标4得分
   - 目标4达成度
   - 总评得分

## 注意事项

1. **权重配置**：确保各课程目标权重总和为1.0
2. **文件格式**：Excel文件必须包含完整的表头和数据
3. **达成度范围**：达成度值应在0-1之间
4. **输出目录**：确保有写入`./output`目录的权限
5. **字体支持**：系统使用中文字体，确保系统已安装SimHei或Microsoft YaHei字体

## 依赖库

系统需要以下Python库：
- pandas
- numpy
- matplotlib
- seaborn
- python-docx
- tkinter（Python标准库）

安装依赖：
```bash
pip install pandas numpy matplotlib seaborn python-docx
```

## 故障排除

### 问题1：导入Excel失败
**原因**：文件格式不正确或缺少表头
**解决**：检查Excel文件格式，确保包含"课程目标"表头

### 问题2：图表中文乱码
**原因**：缺少中文字体
**解决**：安装SimHei或Microsoft YaHei字体

### 问题3：生成Word报告失败
**原因**：缺少python-docx库
**解决**：运行`pip install python-docx`

### 问题4：打印功能无法打开文件
**原因**：报告文件未生成或路径错误
**解决**：先点击"生成Markdown报告"，确保报告生成成功

## 系统优势

相比原始Python脚本，本系统具有以下优势：

1. **用户友好**：图形界面，无需编程知识
2. **通用性强**：可配置不同课程，适用于各种场景
