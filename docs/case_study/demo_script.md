# MAD-KGDT 案例评比 Demo 拍摄脚本

> 评委 5 分钟看不出全貌 — 用 3 段 30 秒 demo 视频抢眼球。
> 录屏工具：OBS Studio / Mac QuickTime；分辨率 1920×1080；30 fps。

---

## Demo 1 · 学生侧（30 秒）

> **核心卖点**：AI 批阅 + 跨设备同步 + 数字孪生

### 镜头脚本

| 时间 | 画面 | 旁白 / 字幕 |
|------|------|-----------|
| 0-3s | Windows 桌面打开 exe，登录页 → 输入学号 | "学生小张打开 MAD-KGDT" |
| 4-8s | 进首页 → 点知识图谱 → 节点交互拖拽 → 点开 "Flutter 动画系统" | "节点拖拽 + 章节定位" |
| 9-14s | 详情页 → 视频教学 → 测验入口 → 答 3 道题 | "学练一体" |
| 15-20s | 转手机 → 错题本同步出现 → 学生提交实验报告 PDF | "跨设备秒级同步" |
| 21-26s | 30 秒后 → 收到通知 "AI 已批阅" → 打开看 4 维评分 + 改进建议 | "AI 批阅秒级反馈" |
| 27-30s | 字幕："88 个学生 / 24 个 Agent / 4 端同步" | 收尾 |

- [ ] 登录页 noir 编辑感（蓝色主题色填充）

## Demo 4 · AI 教学闭环深度（30 秒，Phase 4 新增）

> **核心卖点**：多 Agent 协作批阅 + 班级问答 AI 起草 + 调用全程留痕

### 镜头脚本

| 时间 | 画面 | 旁白 / 字幕 |
|------|------|-----------|
| 0-3s | 教师工作台 → 点 "AI 调用统计" 卡片（紫色 insights 图标） | "全程留痕的 AI 教学" |
| 4-9s | Tab1 Agent 排行榜：tutor / lab_grading / safety… 各自 COUNT + 平均耗时进度条 | "24 Agent 调用频次一目了然" |
| 10-14s | 切 Tab2 调用链路 → 点开一条 chain → 看到 safety→lab_grading→ethics 三步耗时拆解 | "Orchestrator 多 Agent 串联" |
| 15-20s | 回到实验批阅页 → 打开 "增强批阅" Switch → 选 1 份疑似 AI 代写的报告 → 点开始 | "一键三 Agent 协作" |
| 21-25s | 批阅完成 → 反馈区底部展开"🛡 安全审查"+"⚖️ 学术伦理建议"两段 | "学术诚信内置守门人" |
| 26-30s | 回首页 → 进班级问答详情页 → 点 "AI 起草回复" → 草稿写入文本框 → 教师改一句发出 | "AI 起草，教师终审" |

### 录制建议

录这段前**先生成数据**：
1. 学生侧：用增强模式批阅 3 份报告（生成 chain 数据）
2. 学生侧：班级问答发 2 个问题（让 detail 页有内容）
3. 教师侧：让 tutor / lab_grading / safety / ethics 各被调用过 5 次以上（排行榜才好看）

### BGM 建议

- 字体：Source Han Sans 思源黑体 Regular，36px
- 颜色：白底黑字 / 黑底琥珀字
- 位置：底部居中，距底 10%

## 拍摄前置准备

1. **种子数据**：保证演示用的 8 个学生数据齐全（已批阅的实验报告 + 已生成的图谱）
2. **AI Token**：确保 DeepSeek / GLM API Key 有效，余额够 100 次调用
3. **网络**：录屏前禁用学生自动同步 (sync_enabled = false)，避免自动 push 干扰
4. **桌面端 v0.12.0 主题色**：选 "紫蓝渐变" 默认主题录前两段；第三段切到 "深绿" 演示主题切换
5. **画质**：1920×1080 @ 60fps；H.264 编码；输出文件 ≤ 50 MB / 段
