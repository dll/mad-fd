# MAD-KGDT 用户使用手册

> **移动图谱与数字孪生教学系统（MAD-KGDT）** 用户操作手册
> 版本：v0.13.0（2026-05-24）
> 公网 Web：https://dll.github.io/mad-fd/

---

## 第一章 快速开始

### 1.1 获取应用

| 平台 | 获取方式 |
|------|---------|
| **Windows 桌面** | 下载 `移动图谱与数字孪生v0.13.0.exe`，双击运行 |
| **Android 手机** | 安装 `app-release.apk`（132M）|
| **Web 浏览器** | 直接访问 https://dll.github.io/mad-fd/ |
| **HarmonyOS** | 通过 DevEco Studio 构建 HAP（教师演示用，待官方上架）|

### 1.2 首次启动

应用启动后会自动初始化：
- 复制预置数据库（含 6 章课程内容、52 道测验题、知识图谱）
- 后台索引向量 RAG（仅首次约 1-2 分钟）
- 同步 Gitee 远程数据（如已配置）

### 1.3 登录

**默认账号规则**：
- **学号 / 工号** 即用户名（管理员 419116）
- **密码** = 学号末 6 位（管理员 419116）

例：学号 `2023210586` 的密码是 `210586`。

> 密码规则不可改 — 系统统一生成。

**登录方式**：
1. **账号密码**（默认）：输入学号 + 密码 → 点"登录"
2. **快速登录**（设置开启后）：选学生 / 教师 / 管理员一键登录
3. **扫码登录**：桌面端启动服务器后显示 QR 码，手机端扫码加入
4. **语音登录**：点麦克风按钮，说出学号

登录前可点底部 **《用户协议》** / **《隐私声明》** 阅读。

### 1.4 默认进入首页

登录成功后默认进入"首页"Tab（不是图谱）。学生 / 教师 / 管理员看到的菜单不同：

- **学生**：6 Tab — 首页 / 图谱 / 学习 / 实验 / 考核 / 作品
- **教师**：8 Tab — 首页 / 图谱 / 教学 / 课堂 / 实验 / 考核 / 作品 / 达成
- **管理员**：在教师基础上增加"管理"Tab

## 第二章 学生使用指南

### 2.1 首页

进入应用看到的第一屏。包含：
- 欢迎卡片（角色 + 班级信息）
- 学习数据概览（已学时长 / 测验得分 / 错题数）
- 快捷工具网格（图谱 / 测验 / Git仓库 / AI 技能 / 智慧问答 / 我的数据 / ...）
- 最近动态

### 2.2 知识图谱

**操作路径**：底栏 Tab "图谱" → 选章节 → 节点详情

**功能**：
- 拖拽节点查看依赖关系
- 点节点 → 显示概念 / 难度 / 关联资源 / 测验题
- 右上角搜索查具体概念
- ⭐ 收藏感兴趣的节点

### 2.3 学习中心

**操作路径**：底栏 "学习" Tab

**核心功能**：
- 6 章视频教程（点视频卡播放）
- PDF 课件（章节资料）
- 学习进度跟踪
- 学习计划（AI 推荐）
- 薄弱诊断（错题分析 + 补救建议）

### 2.4 章节测验

**入口**：首页"章节测验"卡片 / 或图谱节点详情中的"立即测验"

**流程**：
1. 选章节
2. 答 5-10 道四选一
3. 提交后看正确率 + 错题解析
4. 错题自动加入"错题本"

### 2.5 实验提交

**操作路径**：底栏 "实验" Tab

**6 个实验**：
- 实验 1：开发环境搭建
- 实验 2：原生应用开发
- 实验 3：Flutter 跨平台
- 实验 4：微信小程序
- 实验 5：HarmonyOS 多端
- 实验 6：综合项目实战

**提交步骤**：
1. 点实验卡 → 上传报告 PDF
2. 系统自动同步到 Gitee 远程仓库
3. 教师批阅后会回流分数 + 评语到学生端

### 2.6 班级问答

**入口**：首页菜单"班级问答"

**学生功能**：
- 发问题（私聊老师 / 全班可见）
- 看其他同学的问题与回复
- 点"AI 起草回复" → tutor agent 帮你起草答案 → 编辑后发送
- 标记最佳回答（采纳）

### 2.7 作品展评

**操作路径**：底栏 "作品" Tab

**功能**：
- 提交作品（演示视频 + 源码仓库 + 设计稿）
- 看其他同学作品（前 20% 优秀作品高亮）
- 点赞 / 评论
- 教师评分后看反馈

### 2.8 个人中心

**入口**：右上角头像 → "学生中心"

**功能**：
- 错题本 / 收藏夹 / 学习计划 / 测验记录 / 薄弱诊断 / 我的实验
- **我的数据**（隐私权利）：导出 JSON / 删除本地数据
- **协议声明**：随时查看用户协议与隐私声明

## 第三章 教师使用指南

### 3.1 教师工作台

**入口**：头像菜单 → "教师工作台"

**4 KPI 卡片**：
- 已批阅 / 待批阅 / 平均耗时 / 平均给分

**快捷工具网格**：
- 学生管理 / 题库管理 / 班级管理 / 教学管理 / 实验管理
- 成绩统计 / **AI 调用统计** / 数据导出
- 知识图谱 / 数据同步

### 3.2 实验批阅（重点）

**操作路径**：底栏 "实验" Tab → 切到 "AI 批阅" 子 Tab

**批阅模式**：
1. **标准批阅**：单 Agent（lab_grading），低成本
2. **增强批阅**（Switch 打开）：3 Agent 链路 — safety 安全审查 → lab_grading 主批阅 → ethics 学术伦理建议；适合疑似 AI 代写 / 涉敏内容

**流程**：
1. 选实验任务
2. 点"开始批量 AI 批阅"
3. 批阅结果显示：分数 / 维度评分 / 优点 / 改进点 / AI 代写预警
4. 教师可调整分数 / 编辑反馈
5. 点"批量核准" → 写入 lab_submissions 同步给学生

### 3.3 班级问答管理

**入口**：首页菜单"班级问答（管理）"

**教师功能**：
- 看私聊老师的问题（学生不可见的私问）
- 一键 "AI 起草回复" → tutor agent 起草 → 编辑后发出
- 教师回复自动转 status `answered`
- 学生可标记你的回复为"最佳"

### 3.4 课堂管理

**操作路径**：底栏 "课堂" Tab

**4 子 Tab**：
- 在线状态：实时看哪些学生在线
- 签到管理：发起签到 / 看签到记录
- 课堂互动：发消息 / 提问 / 投票
- 教学工具：举手 / 抽签 / 计时器

### 3.5 考核管理

**操作路径**：底栏 "考核" Tab

**教师 8 子 Tab**（学生 7 个）：
- 分组（多维度查看 — 仓库 / 班组 / 项目 / 角色 / 技术栈 / 特色功能）
- 项目（立项审查）
- 贡献（3 维度评分）
- 材料（PDF / PPT 上传）
- 答辩（排程 + 记录）
- 报告（一键生成 PDF）
- 成绩（多视图统计）
- AI 批阅（仅教师可见）

### 3.6 成绩达成度

**操作路径**：底栏 "达成" Tab

**OBE 工程教育认证支持**：
- 4 课程目标（O1 知识 / O2 实践 / O3 工程 / O4 创新）
- 三维成绩（平时 30% + 实验 30% + 考核 40%）
- 雷达图可视化
- 一键生成 PDCA 持续改进报告
- PDF 输出含封面 + 4 章节 + 评语

### 3.7 AI 调用统计仪表板

**入口**：教师工作台 → "AI 调用统计"

**Tab1：Agent 排行**
- 各 Agent 调用次数 / 平均耗时 / 字符消耗
- 进度条直观对比

**Tab2：调用链路**
- 列出最近 30 条 Orchestrator chain
- 点击展开 → 看 chainStep 0/1/2 各步详情
- 适合分析"增强批阅一次总共耗时多久"

## 第四章 管理员使用指南

### 4.1 管理 Tab

仅管理员可见。包含 14 个工具卡：
- 学生管理 / 教师管理 / 申请审核
- 班级管理 / 教学管理 / 实验管理
- 题库管理 / 问卷管理 / 反馈管理
- 数据导入 / 数据导出 / 仓库分析
- 数据同步 / 多端互通 / 系统设置

**Debug 构建额外**：
- "Demo 数据种子"卡片（紫色）— 一键造演示数据 / 一键撤销

### 4.2 一键生课

**入口**：管理 Tab → 教学管理 → "+" 新课程 → "一键生成"

**流程**：
1. 输入新课程名（如"数据结构"）
2. AI 自动生成：
   - 4 课程目标
   - 8-12 章主题大纲
   - 每章知识图谱节点
   - 200+ 道测验题
   - 6-8 次实验框架
   - 完整达成度评价体系
3. 一键确认入库 → 切换到新课程

### 4.3 数据同步

**Gitee 仓库**：
- 默认仓库：osgisOne/mad-fd
- Token 集中在 `GiteeCredentials.syncToken`
- 学生提交后立即 `unawaited(uploadStudentData)`
- 教师定时拉取 `downloadStudentData(userId)`

**手动同步**：管理 Tab → 数据同步 → 选学生 → "立即同步"

### 4.4 数据导入 / 导出

- 导入：JSON / Excel 批量学生名单
- 导出：全表 JSON 备份 / 成绩单 Excel

## 第七章 隐私与数据

### 7.1 用户协议

**核心约定**：
- 账号 = 学号 / 工号；密码 = 末 6 位（不可改）
- 不得提交他人作品冒充自己（学术诚信）
- 不得用 AI 代写实验报告（safety agent 自动检测 AI_SUSPECT）
- 用户上传内容著作权归本人

### 7.2 隐私声明

**收集**：账号信息 / 学习行为 / 提交内容 / 教学批阅 / AI 调用日志
**不收集**：通讯录 / 短信 / 相册（除非主动上传作业）/ 位置 / IMEI

**存储位置**：
- 本地数据库（sqflite）
- Gitee 仓库（无服务器同步副本）
- LLM 服务商（仅单次调用，不留训练数据）

### 7.3 用户权利

**入口**：个人中心 → "我的数据" / "协议声明"

- **查询权**：随时看自己的全部数据
- **导出权**：一键导出 JSON 文件
- **删除权**：清空本地（教学已核准成绩属档案不删；远程联系管理员）
- **更正权**：通过班级问答 / 反馈申请

## 第八章 常见问题

### 8.1 登录不进去？

- 检查密码是否是学号末 6 位
- 学校是否同步过你的账号（管理员 → 学生管理 → 看名单）
- 网络问题：用本地账号密码登录不需要网络
- 如出"数据库锁定提示" → 关闭其他实例 / 重启应用

### 8.2 同步学生数据失败？

- 检查 Gitee 配置：管理 → 数据同步 → "测试连接"
- Token 是否过期
- 网络是否能访问 https://gitee.com

### 8.3 视频播放空白？

- 桌面端首次播放需要联网下载 libmpv（自动）
- 实验视频源在 `data/视频/` 目录，确认文件存在
- Web 端不支持 mp4 自动播放，用浏览器内置播放器

### 8.4 AI 批阅没反应？

- 检查 AI 配置：设置 → AI 配置 → 测试连接
- DeepSeek / 智谱 token 余额是否充足
- 离线场景可切到 Ollama 本地模型

### 8.5 班级问答 AI 起草等很久？

- 默认调 tutor agent + RAG 检索，约 5-15 秒
- 如长时间无响应 → 检查 AI 配置或换 provider

### 8.6 多端互通连不上？

- 检查双方在同一 WiFi
- 桌面端防火墙是否允许端口（默认 8765）
- HarmonyOS 端：当前版本未真机验证，建议用 Web 端代替

### 8.7 重新构建后窗体标题没变？

按 CLAUDE.md "升版三件套"全套替换 13 处文件（pubspec / main.dart / strings.xml / CMakeLists / main.cpp / Runner.rc / index.html / manifest.json / ohos）。
